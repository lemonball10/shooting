Javascriptの実行

Javascriptを実行するためには、ブラウザを立ち上げて、 「F12」キーで開発者画面を表示した後、「Console」画面を表示する。

Javascriptのルール

文字数と変数

文字列は「"(ダブルクオーテーション)」で挟む。変数は、変数名をそのまま書く

            
文字列の例
console.log("Hello World!");

変数名の例(""で囲まない)
hensu = 10;

Javascript独自のルール

変数の宣言方法

  • let(レット)
    • 変数を宣言する際につける
  • const(コンスト)
    • 定数を宣言する際につける
  • var(バー)
    • 現在は使われていないが、変数を宣言する際に使うことができる 

let a = 5;
const b = 10;
